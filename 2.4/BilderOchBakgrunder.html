Bakgrund

background - Denna egenskap sätter alla bakgrundsegenskaper i en och samma deklaration. De egenskaper som kan sättas är background-color, background-image, background-position, background-size, background-repeat, background-origin, background-clip och background-attachment. Det går bra att utelämna något eller några av värdena. Exempel, med bakgrundsbilder på taggarna body, h1, li och eventuellt flera:

  • Ett
  • Två
  • Tre
  • Fyra

Beskriv och exemplifiera även hur man använder CSS med img-taggen.

Visa hur man gör så att en text flödar (med ett visst avstånd) bredvid en bild så att det inte blir (för stort) tomrum bredvid bilden men om det skulle råka vara en rubrik (h1, h2 och så vidare) i denna text så ska allt (inklusive rubriken) fortsätta under bilden (se egenskapen clear).

Genom att applicera CSS på img-taggen kan man tex. styra hur bilder ska placeras i förhållande till en text. Det går även att lägga till annan formatering som ramar runt bilderna. Genom att använda egenskapen float kan man få bilden att placera sig till vänster (left) eller höger (right) om efterföljande text. Element som följer efter ett flytande element kommer alltså att flyta runt det. Om man vill undvika detta och istället få nästkommande text att placera sig under bilden, kan man använda egenskapen clear.

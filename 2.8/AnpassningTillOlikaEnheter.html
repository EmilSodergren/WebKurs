Sidbaserad regel

@page

Denna regel används för att modifiera vissa CSS-egenskaper för en sida vid utskrift; en såkallad sidbox med ändlig höjd och bredd definieras. Dimensionerna för en sidbox definieras med size-egenskapen. Med margin-egenskapen definieras området som omger sidboxen. Exempel:

@page {
    margin: 2cm;
}

Sidbaserade egenskaper

page-break-after

Den här egenskapen används för att ange att en sidbrytning ska läggas in efter ett visst element. Exempel (vid utskrift ska en sidbryning alltid läggas till efter ett img-element):

@media print {
    img {page-break-after: always;}
}

page-break-before

Den här egenskapen används för att ange att en sidbrytning ska läggas in före ett visst element. Exempel (vid utskrift ska en sidbryning alltid läggas till före ett h2-element):

@media print {
    h2 {page-break-before: always;}
}

page-break-inside

Den här egenskapen används för att ange om en sidbrytning är tillåten inom ett visst element. Exempel (vid utskrift är en sidbryning inte tillåten inom ett ul-element):

@media print {
    ul {page-break-inside: avoid;}
}

size

Den här egenskapen används för att ange storlek och sidorientering för en sidbox. Exempel:

@page {
size: A4 landscape;
}

orphans

Den här egenskapen används för att ange det minsta antalet rader för ett block-element som måste lämnas i slutet av en sida. Exempel:

div {
    orphans: 3;
}

widows

Den här egenskapen används då en paragraf delas upp på två sidor, för att ange det minsta antalet rader som måste lämnas på toppen av den andra sidan. Exempel:

div {
    widiws: 2;
}
